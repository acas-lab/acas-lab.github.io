Memory Architecture

In Memory Architecture, we delve into diverse areas, including DRAM Systems, Heterogeneous Memory Systems, Secure Memory, and Near-data Processing. We passionately investigate how these complex architectures function and interact, aiming to optimize their performance, power efficiency, and security across a wide range of computing tasks. Our work focuses on enhancing DRAM efficiency, creating solutions for future memory systems, and designing innovative near-data processing architectures like Process-in-Memory (PIM).
